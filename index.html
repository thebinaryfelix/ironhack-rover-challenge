Para executar as funções do Rover (usando o Google Chrome):

  1. Execute o jogo, clicando no botão verde
  2. Abra o console (CTRL + SHIFT + I)
  3. Para executar a função moveForward, digite: board.rover.moveForward()
  4. Aperte Enter
